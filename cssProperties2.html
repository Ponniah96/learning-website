CSS units

    Types of CSS units
  1. Absolute Units
  2. Relative Units

Absolute Units

What is absolute units?
Units value doesn't change based on screen resolution
    Absolute Units List
  • cm(centimeter 46px)
  • mm(millimeter 4.6px )
  • in(inch 96px )
  • pt(point 1.33px )
  • pc(picas 1.33*12=16px )

Relative Units

What is relative units?
Units value does change based on screen resolution
    Relative Units List
  • em(element)
  • rem(root element)
  • vw(viewport width)
  • vh(viewport height)
  • %(relative to parent element)
