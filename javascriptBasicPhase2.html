12.Objects
Definition
JavaScript Object is a variable whoch contains multiple values
Javascript objects is a collection of named values and pair, called properties and methods
Ways to create Object
  1. 1.Using Object Literals (Most Preferrable)

    const objName={id:"1",name:"Ponniah", methodName:function(){}}

    const objName={};

    objName.id="1"

    objName.name="Ponniah"

    objName.methodName=function(){}

  2. 2.Using Object new keyword (Non Preferrable)

    const objName=new Object();

    objName.id="1"

    objName.name="Ponniah"

  3. Object.create()

    Object.Create() will create new objects

    const objName={id:"1",name:"Ponniah", methodName:function(){}}

    const newObj=Object.Create(objName)

13.JavaScript Classes
JavaScript classes are special functions which is introduced in ES6
Key Features of javascript classes:
  • class keyword
  • className
  • constructor function
  • methods
  • Extends
  • super
  • static keyword
  • public, private access specifier
Examples
  1. Simple Example:(Add 2 numbers)

    Input Numbers: 2, 3

    Result: 

                                                    
class AddNumbers{
    constructor(num1, num2){
        this.num1=num1;
        this.num2=num2;
    }
    add(){
        console.log('Add: ',this.num1+this.num2);
    }
}
function classAddNumbers(){
    const addNumbers = new AddNumbers(2,3) ;
    addNumbers.add();
}   
classAddNumbers()
  2. Inheritance:(Get Data from parent class)

    Input Numbers: output of parent component and new numbers

    With the help of super constructor, we can parent class properties, methods, value

    Result: 

                                                    
class AddNumbers1{
    constructor(num1, num2){
        this.num1=num1!==undefined? num1 : 3;
        this.num2=num2!==undefined? num2 : 4;
    }
    add(){
        console.log('Parent Add Result: ',this.num1 + this.num2);
        return this.num1 + this.num2;
    }
}
class AddNumbersInheritance1 extends AddNumbers1{
    constructor(num3){
        super();
        this.newNumber=num3;
        this.inheritanceValue= this.add();
    }
    newAdd(){
        console.log('Child Add Result: ',this.newNumber + this.inheritanceValue);
        return this.newNumber + this.inheritanceValue;
    }
}
function classAddNumbersInheritance(){
    const inheritanceClass= new AddNumbersInheritance1(7);
    inheritanceClass.newAdd()
}
