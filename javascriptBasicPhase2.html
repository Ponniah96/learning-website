12.Objects
Definition
JavaScript Object is a variable whoch contains multiple values
Javascript objects is a collection of named values and pair, called properties and methods
Ways to create Object
  1. 1.Using Object Literals (Most Preferrable)

    const objName={id:"1",name:"Ponniah", methodName:function(){}}

    const objName={};

    objName.id="1"

    objName.name="Ponniah"

    objName.methodName=function(){}

  2. 2.Using Object new keyword (Non Preferrable)

    const objName=new Object();

    objName.id="1"

    objName.name="Ponniah"

  3. Object.create()

    Object.Create() will create new objects

    const objName={id:"1",name:"Ponniah", methodName:function(){}}

    const newObj=Object.Create(objName)

Objects values are mutable

If we create an object and assign to another variable like below, if we add value, it will overwrite exisiting property also

const objName={id:"1",name:"Ponniah", methodName:function(){}}

const newObj=objName;

newObj.id="2" //Overwrit objName.id property value also


Object Properties
Object properties contained named values
Objects properties we can add, del, access, modify, change it as read-only
Access Object Properties
  1. Object.PropertyName (obj.id)
  2. Object["PropertyName"] (obj["id"])
Add New Properties

const objName={id:"1",name:"Ponniah", methodName:function(){}}

objName.age="26"

Delete Properties

const objName={id:"1",name:"Ponniah", methodName:function(){}}

delete objName.age

or

delete objName["age"]
