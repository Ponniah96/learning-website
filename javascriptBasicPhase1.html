9.Numbers

9.1.Points to Be Remembered
  1. Numbers are 64-bit Floating Point
    • Js Numbers can display precisely upto 17 decimals
    • Floating Point arithmetic operaions wont work accurate

      Eg: 0.2+0.3 will not work. Instead use (0.2*10 +0.3*10)/10 it will work

  2. Numeric Strings
    • Adding Numers and string will display string oncatenation

      Eg: 10 + "20" =1020(Because + symbol considered as string concatenation operator)

      But 10+20+"30"=3030

      x=10, y=20 , document.innerHTML= x+y ==> 1020 (Because it will do as string concatenation)

    • All Artimetic operations between strings are possible except addition

      "10" * "20"=200

      "20" - "10"=10

      "20" / "10"=2

      "20"+"10"=1020

  3. NaN operator (Not a Number)
    • It will occur on below scenarios

      10 * "Hi"

      2 / 0

    • NaN operator

      It will display true or false if resultant value is number or not

      x=10*"Hi". isNaN(x)==>True

      x=10*2. isNan(x)==>false

      x=NaN+5 //x is Nan

    • typeofNaN is number
  4. Infinity
    • if numbers are beyond limitations then infinity wil come
    • typeof(infiniy)= numbers because it is also number

11.4.Array Iteration
  1. ForEach Function

    ForEach Function calls function at each element of the array

    Syntax: Array.forEach(value,item,array)

      Definitions:
    • Value: Describes individual array element
    • Item: Describes index of array element
    • Array: Describes Entire array

    Array: [10,20,30,40,50]



      • Map Function

        Map Function calls function at each element of the array and store resultant operation in new array

        Syntax: const newArray = Array.map(value,item,array)

          Definitions:
        • Value: Describes individual array element
        • Item: Describes index of array element
        • Array: Describes Entire array

        Array: [10,20,30,40,50]


        • Filter Function

          Filter Function returns new array which contains specific exisiting array elements only if we met specific criteria

          Syntax: const newArray= Array.filter(value,item,array)

          Array : [10,20,30,40,50]


          • Reduce Function

            Reduce Function reduce entire array into single value(Number/Strings) based on operations done between array elements

            Syntax: const newArray = Array.map(total,value,item,array)

              Definitions:
            • Total: Describes Intial or previous result value
            • Value: Describes individual array element
            • Item: Describes index of array element
            • Array: Describes Entire array

            Array: [10,20,30,40,50]


          • ReduceRight Function

            Reduce Right Function reduce entire array into single value(Number/Strings) based on operations done between array elements starting from right to left

            Syntax: const newArray = Array.map(total,value,item,array)

              Definitions:
            • Total: Describes Intial or previous result value
            • Value: Describes individual array element
            • Item: Describes index of array element
            • Array: Describes Entire array

            Array: ['10','20','30','40','50']
